## ✨ Présentation

**DEV SHADOW-TECH** est un bot WhatsApp userbot ultra-puissant développé entièrement sur téléphone par **SHADOW 🌸 TECH 🌸**. Avec plus de **50 commandes**, il est conçu pour gérer, animer et protéger vos groupes WhatsApp.

```
🤖 Bot  : Userbot (numéro WhatsApp réel)
⚡ Base  : Baileys + Node.js
📦 Cmds : 50+ commandes
🌍 Cible : Afrique francophone
```

## 📦 Commandes

### 👥 GROUPE (12 commandes)
| Commande | Description |
|----------|-------------|
| `.antilink on/off` | Active/désactive l'antilink |
| `.tagall` | Mentionne tous les membres |
| `.tagadmin [message]` | Mentionne tous les admins |
| `.kick @user` | Expulse un membre |
| `.add numéro` | Ajoute un membre au groupe |
| `.promote @user` | Promeut un membre en admin |
| `.demote @user` | Rétrograde un admin |
| `.grouppp` | Change la photo du groupe |
| `.welcome on/off` | Active le message de bienvenue |
| `.goodbye on/off` | Active le message d'au revoir |
| `.mute` | Réduit le groupe au silence |
| `.unmute` | Réactive les messages du groupe |

### 🎬 MÉDIA (2 commandes)
| Commande | Description |
|----------|-------------|
| `.sticker` | Convertit une image en sticker |
| `.toimg` | Convertit un sticker en image |

### 📥 TÉLÉCHARGEMENTS (1 commande)
| Commande | Description |
|----------|-------------|
| `.mp3 [titre ou lien]` | Télécharge un audio YouTube |

### 🎮 FUN & JEUX (5 commandes)
| Commande | Description |
|----------|-------------|
| `.quiz` | Lance un quiz interactif |
| `.idee` | Génère une idée fun aléatoire |
| `.citation [vie/succes/motivation/islam/amour]` | Citation inspirante |
| `.autoreact on/off` | Réagit automatiquement aux messages |
| `.jeu` | Mini-jeu amusant |

### 🔧 GÉNÉRAL (10 commandes)
| Commande | Description |
|----------|-------------|
| `.menu` | Affiche toutes les commandes |
| `.ping` | Vérifie si le bot est actif |
| `.owner` | Infos sur le créateur |
| `.mode public/private` | Change le mode du bot |
| `.info` | Infos sur le bot |
| `.groupe` | Lien du groupe officiel |
| `.chaine` | Lien de la chaîne officielle |
| `.aide [commande]` | Aide sur une commande |
| `.uptime` | Temps de fonctionnement |

### 👑 OWNER (5 commandes)
| Commande | Description |
|----------|-------------|

## 🚀 Installation

### Prérequis
```
✅ Node.js 18+
✅ npm
✅ Un numéro WhatsApp actif
✅ Un compte sur un hébergeur
```

### Étapes

**1. Clone le repository**
```bash
git clone https://github.com/guthubc-collab/AIGER-XMD.git
cd AIGER-XMD
```

**2. Installe les dépendances**
```bash
npm install
```

**3. Configure le `.env`**
```bash
cp .env.example .env
nano .env
```

**4. Lance le bot**
```bash
npm start
```

**5. Connecte ton numéro**
```
📲 WhatsApp → Appareils liés
→ Lier avec numéro de téléphone
→ Entre le code qui s'affiche dans le terminal
```
